Exercise

Create a flexible horizontal navigation menu using display:flex. Take the following requirements into account:

  • The menu items should be evenly distributed over the total width of header
  • Not only the text of a menu-item should be clickable, also the (invisible) space around the text. In fact every click in the overlay zone should lead to a webpage.
  • When a menu item is hovered, the item is highlighted by a border.
  • Valid CSS.
  • Do you prefer a smooth transition or more subtle effects? Try to implement it yourself, take a look at https://www.w3schools.com/css/css3_transitions.asp.

Home

These exercises focus on ‘home.html’.

  • (documentation) Use display: grid to position the main content (text: “This is Leuven”) and the aside (“Events”). The main content uses two third of the available width. Between main and aside, there is an empty space of 20px. Add the necessary div's and id's. REMARK In the example of the documentation, both header and footer are part of the grid. This is abundant and should be omitted.
  • Use display: grid to position both events in the aside. Don't add any code to the html-document. Use the existing id's, classes, ... The title (h2) takes a height of 4em. There is a gap of 20px between both columns.
    (REMARK) When you make your own project, you can make the positioning even more simple by adding an extra div around the two events.
